| Purpose               | Command                            |
| --------------------- | ---------------------------------- |
| Compile TypeScript    | `npx tsc`                          |
| Watch mode            | `npx tsc --watch`                  |
| Check for type errors | `npx tsc --noEmit`                 |
| Run .ts file directly | `npx ts-node src/test.ts`          |
| Clean dist folder     | `rm -rf dist` or `npx rimraf dist` |
